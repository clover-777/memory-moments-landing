# Memory Moments - Фотопечать на чашках и футболках

## 📋 Описание

Модерный лендинг-пейдж фотоцентра с интерактивным конструктором для создания макетов печати на чашках и футболках.

## ✨ Возможности

### Конструктор макетов
- ✅ Выбор изделия (чашка или футболка)
- ✅ Выбор цвета изделия (белый, черный, синий, красный, зеленый)
- ✅ Загрузка собственного фото
- ✅ Добавление текста с настройкой размера и цвета
- ✅ Превью в реальном времени
- ✅ Скачивание готового макета

### Секции сайта
- 🏠 Героический баннер с призывом к действию
- 📋 Описание услуг
- 🖼️ Галерея примеров
- 🎨 Интерактивный конструктор
- 📞 Контактная информация

## 🚀 Быстрый старт

### Локально
```bash
# Клонируйте репозиторий
git clone https://github.com/clover-777/memory-moments-landing.git

# Перейдите в папку проекта
cd memory-moments-landing

# Откройте index.html в браузере
# Или используйте Live Server в VS Code
```

### На GitHub Pages
1. Перейдите в Settings репозитория
2. Найдите секцию "Pages"
3. Выберите ветку "main" как источник
4. Сайт будет доступен по адресу: `https://clover-777.github.io/memory-moments-landing/`

## 💡 Как использовать конструктор

1. **Выберите изделие**: чашка или футболка
2. **Выберите цвет**: нажмите на цвет изделия
3. **Загрузите фото**: кликните на поле загрузки и выберите изображение
4. **Добавьте текст** (опционально):
   - Введите текст
   - Отрегулируйте размер и цвет
   - Нажмите "Добавить текст"
5. **Скачайте макет**: нажмите кнопку "Скачать макет"
6. **Отправьте заказ**: используйте форму контактов или позвоните

## 🛠️ Внесение изменений

### Изменение контактной информации
Отредактируйте секцию `#contacts` в `index.html`:

```html
<a href="tel:+79991234567">+7 (999) 123-45-67</a>
<a href="mailto:info@memorymoments.ru">info@memorymoments.ru</a>
```

### Добавление своих цветов
Добавьте классы в `styles.css`:

```css
.mug-purple { background-color: #9333ea; }
.tshirt-purple { background-color: #9333ea; }
```

Теги в HTML:
```html
<button class="color-btn" data-color="purple" style="background-color: #9333ea;"></button>
```

### Замена примеров в галерее
Замените URL изображений в `index.html`:

```html
<img src="https://ваша-ссылка-на-изображение.jpg" alt="Пример 1">
```
